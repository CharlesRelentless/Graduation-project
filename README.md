# Graduation-project
📚 毕业设计辅助资源库
欢迎使用本毕业设计辅助资源库！本仓库旨在为大学生毕业论文写作提供实用工具与参考资料，涵盖文献管理、论文格式规范、图表制作等内容。

⚠️ 注意：请遵守相关软件版权规定，仅用于学习与科研用途。

🔧 使用建议
✅ 文献管理推荐流程
安装 EndNoteX9软件 并应用 EndNote汉化补丁
使用 Endnote 中国科学技术大学正版授权 进行激活
参考 知网导入Endnote.docx 将 CNKI 文献导入 EndNote
利用 EndNote 管理参考文献并自动生成引用格式
✅ 论文写作支持
阅读 论文陈述观点的（三步法）.pdf 提升逻辑表达能力
使用 论文多级列表.txt 和 三线表.txt 规范排版格式
在撰写技术路线时，可使用 亿图图示激活... 绘制专业图表
