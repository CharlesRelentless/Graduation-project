## 🌟 重点强调：为什么“多级列表”和“三线表”如此重要？

### ✅ 多级列表
- 是 **自动生成目录** 的基础。
- 保证章节层级清晰，符合学校格式要求。
- 避免手动编号错误，提高修改效率。

> 📌 建议：在开始写作前就设置好样式和多级列表，后续只需调整内容即可自动更新目录。

### ✅ 三线表
- 是 **国家自然科学基金、高校论文格式** 的通用标准。
- 表格简洁、专业，体现学术严谨性。
- 避免使用竖线、斜线、阴影等非标准元素。

> 📌 建议：使用 `三分钟快速制作论文三线表.mp4` 学习快捷方法，或参考 `论文多级列表.txt` 获取模板。

## 📌 使用建议

1. **先看视频**：建议按顺序观看 `论文格式` 文件夹下的 6 个视频，系统掌握排版技能。
2. **优先设置样式**：在写正文前完成样式与多级列表配置。
3. **善用 EndNote**：尽早导入文献，避免后期格式混乱。
4. **图表规范**：技术路线图可用亿图图示绘制，确保美观专业。
5. **定期备份**：论文初稿完成后及时保存与备份。
